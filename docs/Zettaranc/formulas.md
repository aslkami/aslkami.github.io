### 指标公式

- 知行中期多空短线
  ```shell
    知行中期多空短线:EMA(EMA(C,10),10),COLORGREEN,LINETHICK1;
    MA1:MA(CLOSE,60);
    MA2:=EMA(CLOSE,13);
    A1:=REF(O,1);
    A2:=ABS((REF(C,1)-A1)/A1);
    A3:=ABS((REF(H,1)-A1)/A1);
    A4:=ABS((REF(L,1)-A1)/A1);
    B1:=ABS((A2+A3+A4)/3);
    B2:=O*B1;
    预测低:=O-B2,LINETHICK0;
    最低价:=LOW,COLORYELLOW,LINETHICK0;
    BBI:(MA(CLOSE,M1)+MA(CLOSE,M2)+MA(CLOSE,M3)+MA(CLOSE,M4))/4,COLORWHITE;
  ```
- 单针 20

  ```shell
    N1:=3;
    N2:=21;
    短期:100*(C-LLV(L,N1))/(HHV(C,N1)-LLV(L,N1)),COLORWHITE;
    中期:=100*(C-LLV(L,10))/(HHV(C,10)-LLV(L,10)),COLORYELLOW;
    中长期:=100*(C-LLV(L,20))/(HHV(C,20)-LLV(L,20)),COLORMAGENTA;
    长期:100*(C-LLV(L,N2))/(HHV(C,N2)-LLV(L,N2)),COLORRED,LINETHICK2;
    四线归零买:=IF((短期<=6 AND 中期<=6 AND 中长期<=6 AND 长期<=6),-30,0),STICK,COLOR0000FF,LINETHICK3;
    白线下20买:=IF(短期<=20 AND 长期>=60,-30,0),STICK,COLOR00FFFF,LINETHICK3;
    白穿红线买:=IF(((CROSS(短期,长期)AND 长期<20)),-30,0),STICK,COLOR00FF00,LINETHICK3;
    白穿黄线买:=IF(((CROSS(短期,中期)AND 中期<30)),-30,0),STICK,COLORFF9150,LINETHICK3; 80,COLORYELLOW;
  20,COLORYELLOW;
  ```

- 自定义 KDJ

  ```shell
    VAR1:=HHV(HIGH,N);
    VAR2:=LLV(LOW,N);
    RSV:=(CLOSE-VAR2)/(VAR1-VAR2)*100;
    K:SMA(RSV,M1,1);
    D:SMA(K,M2,1);
    J:3*K-2*D;
    0,COLORCYAN;
    100,COLORRED;
  ```

### 选股公式

- 多空选股

  ```shell
    市值:=FINANCE(40);
    市值大于50:=市值>5000000000;
    EMA5:=EMA(C,5);
    EMA10:=EMA(C,10);
    EMA上穿:=CROSS(EMA5,EMA10);
    EMA上穿 AND 市值大于50;
  ```

- 严格多空

  ```shell
    市值:=FINANCE(40);
    市值大于50:=市值>5000000000;
    成交量放大:=VOL>MA(VOL,10);
    RSV值:=(C-LLV(L,9))/(HHV(H,9)-LLV(L,9))*100;
    K值:=SMA(RSV值,3,1);
    D值:=SMA(K值,3,1);
    J值:=3*K值-2*D值;
    前10日J小于0:=COUNT(J值<=12,15) >=1;
    EMA5:=EMA(C,5);
    EMA10:=EMA(C,10);
    EMA上穿:=CROSS(EMA5,EMA10);
    EMA上穿 AND 市值大于50 AND 成交量放大 AND 前10日J小于0;
  ```

- J 大负值

  ```shell
    J:=12;
    R:=21;
    N1:=3;
    N2:=21
    RSV值:=(C-LLV(L,9))/(HHV(H,9)-LLV(L,9))*100;
    K值:=SMA(RSV值,3,1);
    D值:=SMA(K值,3,1);
    J值:=3*K值-2*D值;
    短期:=100*(C-LLV(L,N1))/(HHV(C,N1)-LLV(L,N1)),COLORBLUE;
    长期:=100*(C-LLV(L,N2))/(HHV(C,N2)-LLV(L,N2)),COLORRED,LINETHICK2;
    LC:=REF(CLOSE,1);
    自定义RSI:=SMA(MAX(CLOSE-LC,0),N3,1)/SMA(ABS(CLOSE-LC),N3,1)*100;
    红白线接近零:=长期<10 AND 短期<10;
    红白线:=短期<=长期 OR 红白线接近零;
    J值<J AND 自定义RSI<R;
  ```

- 选大蓝筹白马

  ```shell
  { 选股公式：市值大于800亿，基本面稳定，ROE高，现金流充足 }
  { 1. 市值 > 800亿 }
  市值条件 := FINANCE(41) / 10000 > 800;

  { 2. 基本面稳定：连续两年净利润正增长 }
  净利润今年 := FINANCE(27) > 0;           { 最近年度净利润 > 0 }
  净利润去年 := FINANCE(59) > 0;           { 上一年度净利润 > 0 }
  利润增长 := FINANCE(34) > 0;             { 净利润同比增长率 > 0 }
  基本面稳定 := 净利润今年 AND 净利润去年 AND 利润增长;

  { 3. ROE > 15% }
  高ROE := FINANCE(37) > 15;               { 加权净资产收益率 > 15% }

  { 4. 现金流动充足 }
  {    - 每股经营现金流 > 0 }
  {    - 现金流保障倍数（经营现金流/净利润）> 0.7 }
  每股现金流 := FINANCE(44) > 0;           { 每股经营现金流 }
  现金流保障 := IF(FINANCE(27) > 0, FINANCE(24) / FINANCE(27) > 0.7, 1);
  { 若净利润为负则跳过保障倍数检验 }
  现金充足 := 每股现金流 AND 现金流保障;

  { 综合选股条件 }
  选股: 市值条件 AND 基本面稳定 AND 高ROE AND 现金充足;
  ```
